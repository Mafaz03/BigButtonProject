ReInkPy
=======
Waste ink counter resetter for some (EPSON) printers.

Free and open. [Python](https://python.org) code.


# Getting started

```
git cloe https://github.com/Mafaz03/BigButtonProject
```

```
python3 ui.py
```

Works only with usb connected epson printers
